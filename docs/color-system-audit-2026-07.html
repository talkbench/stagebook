Decisions & hardcodes this section forces

  • Point reuse is a choice to ratify. A point can't currently be themed apart from range handles. Fine if intended (a point = a zero-width range) — but document it, or add --stagebook-timeline-point* if points should ever look distinct.

4c · Playhead hue — why rose "sticks out"

The palette's chromatic hues at equal size. Rose is actually a smart pick — it's the one warm slot not claimed by danger-red or warning-amber, so it stays distinct from both. What you can't put a finger on is that rose-600 is the highest-chroma and lightest swatch here, so it reads "louder" than its neighbors. It's a saturation/brightness mismatch, not the hue — so the fix is to tune it to the same weight as the other steps (e.g. rose-700), not to change the color.

5 · Status counters — colored text vs pill

The TextArea char counter (0.75rem) encodes state as text color today — exactly where small-text contrast bites: valid-green is only ~3.3:1 and already fails AA. A pill flips it to dark text on a soft tint (contrast trivial), makes it more visible, and — if the fill only appears off-default — signals state by shape too, not color alone (WCAG 1.4.1). Needs a {fg,bg} pair per state, like --danger/--danger-bg already are.

5c · Semantic {fg, bg} pairs — completing the set

danger already ships as a pair; success & warning need a tint added so pills/callouts have dark-text-on-soft-tint for every tone. Each fg is the smallest ramp step that still passes ≥4.5:1 as text on its own bg — perceptual, so the step differs by hue (red-600, green-700, amber-700). danger got lucky: red-600 is dark enough to be both accent and text; green-600 wasn't, hence the bump.

These three {fg,bg} pairs are exactly what a StatusMessage (chip + callout) consumes — the follow-up issue's dependency on this palette work.

5d · Colorblindness (CVD) — a separate axis from contrast

Passing WCAG contrast (a luminance ratio) says nothing about hue discriminability. Each key color below is simulated under the three color-vision deficiencies (Machado 2009 matrices, full severity). Watch the semantic trio: under red-green CVD, success-green / warning-amber / danger-red drift toward each other. That's why meaning is never carried by color alone (WCAG 1.4.1) — the counter keeps its number, pills keep text, errors keep a message, the playhead is a shape+position.
